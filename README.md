# Architecture

This repository contains reference architecture artifacts and patterns used for documenting, sharing and rendering architecture diagrams.

Top-level folders
- `anti_patterns` — Examples of common architectural anti-patterns.
- `system_designs` — System-level DSLs and diagrams (each system typically has its own subfolder).
- `platform_engineering` — Infrastructure and platform patterns and templates.
- `cloud_architecture` — Cloud-specific reference designs and patterns.
